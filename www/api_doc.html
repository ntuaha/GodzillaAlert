當前所有事件概況

可以了解當前資料庫內所有事件的持續狀態與更新情形。

GET | http://104.46.234.139/GA/api/getDataInfo.php?beta=T

參數：

GET beta=T 開發中版本

資料內容：

標題 描述
type 事件類型
status 事件狀況
cnt 事件數量
update_dt 更新時間

資料範例：


{
  "data":[
    {
      "type":"警察廣播電台_災變",
      "status":"持續中",
      "cnt":"97",
      "update_dt":"2015-01-10 04:30:04"
    },
    {
      "type":"警察廣播電台_災變",
      "status":"結束",
      "cnt":"4",
      "update_dt":"2015-01-04 01:08:05"
    }
  ]
}

取得當前發生事件

可以取得當前發生所有事件的詳細情況與內容。

GET | http://104.46.234.139/GA/api/getEvent.php?beta=T

參數：

GET beta=T 開發中版本

資料內容：

標題 描述
status 事件狀況
title 事件標題
description 事件描述
update_dt 更新時間
start_dt 起始時間

資料範例：

當前限定數量事件

可以取得當前資料庫內限定數量（最重要 N 個）事件的持續狀態與更新情形。

GET | http://104.46.234.139/GA/api/getImpEvent.php?beta=T&limit=2

參數：

GET beta=T 開發中版本 GET limit=[N] 限定事件數量

資料內容：

標題 描述
type 事件類型
status 事件狀況
title 事件標題
description 事件描述
update_dt 更新時間
start_dt 起始時間
s 資料來源
sources 來源列表

資料範例：


{
  "data":[
    {
      "type":"警察廣播電台_其他",
      "status":"持續中",
      "title":" 南部 [中山高速公路] 南下 ",
      "description":"王田路段.客運任意變換車道+貼近前車.請小心",
      "update_dt":"2015-01-10 23:10:04",
      "start_dt":"2015-01-10 22:05:00",
      "s":"警察廣播電台"
    },
    {
      "type":"警察廣播電台_事故",
      "status":"持續中",
      "title":" 北部 [臺北市] 忠孝橋(往三重方向)",
      "description":"機車道有事故.救護車已達處理.因佔用外側影響車流.回堵至重慶南路",
      "update_dt":"2015-01-10 23:10:04",
      "start_dt":"2015-01-10 22:08:00",
      "s":"警察廣播電台"
    }
  ],
  "sources":[
    "警察廣播電台",
    "中央氣象局"
  ]
}
